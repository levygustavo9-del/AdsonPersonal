⚡ 3 PASSOS PARA COMEÇAR (10 MINUTOS)

1️⃣ INSTALAR (que leva 3-5 minutos)
   
   Abra PowerShell nesta pasta (Adson) e digite:
   
   npm install
   
   (Aguarde até terminar, só uma vez)

2️⃣ EXECUTAR LOCALMENTE

   Depois que terminar, digite:
   
   npm start
   
   A página abre em http://localhost:3000

3️⃣ CONFIGURAR SEU WHATSAPP

   Abra: src/components/Modal.js
   
   Procure pela linha ~95:
   const whatsappNumber = '5511999999999';
   
   Mude '5511999999999' para seu número
   (Formato: 55 + código área + 9 dígitos, sem caracteres especiais)
   
   Exemplo: Seu número é (85) 98765-4321
   Mude para: 5585987654321

❓ DÚVIDAS FREQUENTES

P: Preciso saber programar?
R: NÃO. Você só precisa editar textos em arquivos .js

P: Qual é o custo?
R: Hospedagem GRÁTIS em Netlify. Domínio ~R$40/ano opcional.

P: Funciona sem internet?
R: NÃO. Precisa de internet para WhatsApp funcionar.

P: Posso mudar cores?
R: SIM, em src/App.css. Instruções em CONFIGURACAO.md

P: O WhatsApp funciona mesmo automático?
R: SIM! Testei. A página abre WhatsApp Web com mensagem pronta.

P: Como recebo as mensagens?
R: No seu WhatsApp normal. A página envia sem salvar em servidor.

P: Preciso de projeto Node.js antes disso?
R: NÃO. O projeto já vem com package.json. Só instale npm.
